# naver_webBrowser
NAVER SEARCH GUI CUSTOMIZED WEB BROWSER

* 프로젝트명 : 네이버 검색 포털 만들기
* 사용언어 : python3.9
* 사용툴 : Pycharm, QtDesigner
* 사용라이브러리 : pyqt5, time, datetime, threading, json, webbrowser, urllib, sys, os
* 핵심기술 : 네이버 검색 OPEN API를 활용한 네이버 웹 크롤링 + PYQT5를 활용한 gui 구축
